    <h1>C Language Notes</h1>
    <p>The C programming language is a powerful, efficient, and flexible language that is widely used for system and application software. It was developed in the early 1970s by Dennis Ritchie at Bell Labs.</p>

    <h2>Key Features</h2>
    <ul>
        <li>Low-level access to memory</li>
        <li>Simple set of keywords</li>
        <li>Clean style</li>
        <li>Efficient performance</li>
    </ul>

    <h2>Basic Syntax</h2>
    <p>Here is a simple example of a C program:</p>
    <pre><code>
#include &lt;stdio.h&gt;

int main() {
    printf("Hello, World!\n");
    return 0;
}
    </code></pre>

    <h2>Data Types</h2>
    <p>C supports several data types, including:</p>
    <ul>
        <li><code>int</code> - Integer type</li>
        <li><code>float</code> - Floating point type</li>
        <li><code>char</code> - Character type</li>
        <li><code>double</code> - Double precision floating point type</li>
    </ul>

    <h2>Control Structures</h2>
    <p>C provides several control structures for decision making and looping:</p>
    <ul>
        <li><code>if</code> and <code>else</code> statements</li>
        <li><code>for</code> loops</li>
        <li><code>while</code> loops</li>
        <li><code>switch</code> statements</li>
    </ul>
